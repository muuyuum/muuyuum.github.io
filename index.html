258. Add Digits

Today’s problem is a digit quiz.

For example, we assume number has 5 digits number.
$num = (a + b + c + d + e) + (a \times 9999 + b \times 999 + c \times 99 + d \times 9)$.

and then $a \times 9999 + b \times 999 + c \times 99 + d \times 9$ must be divided by 9.
So the result $num \bmod 9$ is same as $a + b + c + d + e \bmod 9$.

$$f(n) =
\begin{cases}
0 & \quad \text{if } n = 0\\
9 & \quad \text{if } n \neq 0, n \equiv 0 \ (\bmod 9)\\
n \bmod 9 & \quad \text{if } n \not\equiv 0 \ (\bmod 9)\\
\end{cases}
$$
For one fomula is:
$f(n) = (n - 1) \bmod 9 + 1$.

sublime symlink disappeared in OS X El Capitan

Today when my want to use “subl” in terminal, but there is a error message:

-bash: subl: command not found

This is the first time run sublime text after upgrading to El Capitan.
But I think it may have some trouble about my system path.

And try this:
sudo ln -s "/Applications/Sublime Text 2.app/Contents/SharedSupport/bin/subl" /usr/bin/subl

result:
ln: /usr/bin/subl: Operation not permitted

hmmmm….

In El Capitan, user are not allowed to write to /usr/bin, but you can write to /usr/local/bin.
So if you want to add the others path variable, it also should be written in usr/local/bin.

And you can try to do this:
sudo ln -s /Applications/Sublime\ Text\ 2.app/Contents/SharedSupport/bin/subl /usr/local/bin/subl

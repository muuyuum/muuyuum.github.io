292. Nim Game

In one move, you can remove 1~3 stones.
Just think these cases:

1stone: first player remove all (first win)
2stones: first player remove all (first win)
3stones: first player remove all (first win)
4stones: no matter first player remove 1~3, second player remove all (second win)
5stones: first player remove 1, no matter second player remove 1~3, first player remove all (first win)
…

Finally you can find the first win condition that the stones number is multiple of 4.
